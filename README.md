# swiss-knife

All kinds of tools improve your coding experience.

## Introduction

~~~shell
.
├── LICENSE
├── Makefile
├── README.md
└── terminal-translator # As you see, support English and Chinese so far
~~~

## Prequisties

- Go >= 1.25.6
- GNU make

## Usage

~~~sh
# build
make
# install, default in /usr/local/bin, you can specify the location by passing `prefix` variable e.g `make prefix=~/opt/ install`
make install
# help
make help
~~~

Enjoy it!
